# Task Manager

A task management app with React & TypeScript. Manage tasks with authentication and dashboard.

## Features

- Create, edit, delete tasks
- Dashboard with stats
- User authentication
- Task filtering
- Responsive UI

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Setup environment:
   ```bash
   cp .env.example .env
   ```

3. Run development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components/` - UI components
- `src/context/` - Auth state
- `src/hooks/` - Custom hooks
- `src/pages/` - App pages
- `src/types.ts` - TypeScript types
- `tailwind.config.js`: Tailwind CSS configuration.
